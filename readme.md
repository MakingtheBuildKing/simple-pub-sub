# Prequisite

ROS Kinetic Kame


# Getting Started

* First, turn on ros core

    ```bash
    roscore
    ```

* Create another terminal (ctrl + alt + t)

    ```bash
    rosrun tutorial_0 tutorial_0_publisher 
    ```
* Again, create another terminal (ctrl + alt + t)
    ```bash
    rosrun tutorial_0 tutorial_0_subscriber
    ```
